# src

Центральный пакет исходников приложения.

Содержимое:
- `app.py` — точка входа: инициализация логирования, выбор и запуск Qt `MainWindow`.
- `config.py` — централизованная конфигурация: ProcessingConfig, MetricsConfig, UIConfig, пресеты.
- `processing/` — вычислительные пайплайны, кодеки, метрики качества.
  - `transforms/` — реализации методов: FFT, DCT, DWT, FWHT, Huffman-like, Rosenbrock-like
- `ui_new/` — актуальная реализация пользовательского интерфейса на PySide6.
- `utils/` — служебные утилиты (логирование и пр.).

Использование:
- В режиме разработки запускайте `python -m src.app`.
- В собранной версии запускается исполняемый файл из `dist/`.

> **Примечание:** Устаревшие модули (`app_ui/`, `ui/`, `ui2/`) были мигрированы в `ui_new/`.
