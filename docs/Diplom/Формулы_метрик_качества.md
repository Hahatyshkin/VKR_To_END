# Формулы метрик качества аудиосигнала

Все формулы соответствуют реализации в файле `AudioAnalyzer/src/processing/metrics.py`.

Обозначения:
- `x[n]` — опорный (оригинальный) сигнал
- `y[n]` — тестовый (обработанный) сигнал
- `N` — длина сигнала (общая часть)
- `X[k]` — спектр опорного сигнала
- `Y[k]` — спектр тестового сигнала
- `sr` — частота дискретизации

---

## 1. SNR (Signal-to-Noise Ratio)

**Функция:** `compute_snr_db(reference, test)`
**Единица:** дБ | **Направление:** выше — лучше

```
noise[n] = x[n] - y[n]

P_signal = (1/N) * sum(x[n]^2) + eps

P_noise = (1/N) * sum(noise[n]^2) + eps

SNR = 10 * log10(P_signal / P_noise)    [дБ]
```

---

## 2. RMSE (Root Mean Square Error)

**Функция:** `compute_rmse(reference, test)`
**Единица:** относительная | **Направление:** ниже — лучше

```
e[n] = x[n] - y[n]

RMSE = sqrt( (1/N) * sum(e[n]^2) )
```

---

## 3. SI-SDR (Scale-Invariant Signal-to-Distortion Ratio)

**Функция:** `compute_si_sdr_db(reference, test)`
**Единица:** дБ | **Направление:** выше — лучше

```
alpha = sum(x[n] * y[n]) / (sum(x[n]^2) + eps)

y_hat[n] = alpha * x[n]          (целевой компонент)

e_noise[n] = y[n] - y_hat[n]     (шумовая компонента)

SI-SDR = 10 * log10( (sum(y_hat[n]^2) + eps) / (sum(e_noise[n]^2) + eps) )    [дБ]
```

---

## 4. LSD (Log-Spectral Distance)

**Функция:** `compute_lsd_db(reference, test, sr_ref, sr_test, n_fft=1024, hop=512)`
**Единица:** дБ | **Направление:** ниже — лучше

Для каждого окна `m` (с весовой функцией Ханна):

```
X_m[k] = FFT( x_m[n] * w[n] )         (rFFT)
Y_m[k] = FFT( y_m[n] * w[n] )

S_x[k] = 10 * log10( |X_m[k]|^2 + eps )    (лог-амплитуда)
S_y[k] = 10 * log10( |Y_m[k]|^2 + eps )

LSD_m = sqrt( (1/K) * sum( (S_x[k] - S_y[k])^2 ) )

LSD = (1/M) * sum(LSD_m)    [усреднение по окнам, дБ]
```

Где:
- `K = N_fft / 2 + 1` — число частотных бинов
- `M` — число окон
- `w[n]` — окно Ханна: `w[n] = 0.5 * (1 - cos(2*pi*n / (N-1)))`

---

## 5. Spectral Convergence

**Функция:** `compute_spectral_convergence(reference, test, sr_ref, sr_test, n_fft=1024, hop=512)`
**Единица:** относительная | **Направление:** ниже — лучше

Для каждого окна `m`:

```
X_m[k] = rFFT( x_m[n] * w[n] )
Y_m[k] = rFFT( y_m[n] * w[n] )

SC_m = || |X_m| - |Y_m| ||_2 / ( || |X_m| ||_2 + eps )

SC = (1/M) * sum(SC_m)
```

Где:
- `|| |X_m| - |Y_m| ||_2 = sqrt( sum( (|X_m[k]| - |Y_m[k]|)^2 ) )`

---

## 6. Spectral Centroid Difference

**Функция:** `compute_spectral_centroid_diff_hz(reference, test, sr_ref, sr_test, n_fft=1024, hop=512)`
**Единица:** Гц | **Направление:** ниже — лучше

Для каждого окна `m`:

```
X_m[k] = |rFFT( x_m[n] * w[n] )|

df = sr / N_fft                          (частотное разрешение)

centroid_x = sum( k * |X_m[k]| ) / ( sum( |X_m[k]| ) + eps ) * df    [Гц]
centroid_y = sum( k * |Y_m[k]| ) / ( sum( |Y_m[k]| ) + eps ) * df    [Гц]

d_m = |centroid_x - centroid_y|

SC_diff = (1/M) * sum(d_m)    [Гц]
```

---

## 7. Spectral Cosine Similarity

**Функция:** `compute_spectral_cosine_similarity(reference, test, sr_ref, sr_test, n_fft=1024, hop=512)`
**Единица:** 0..1 | **Направление:** выше — лучше

Для каждого окна `m`:

```
A_m[k] = |rFFT( x_m[n] * w[n] )|
B_m[k] = |rFFT( y_m[n] * w[n] )|

cos_m = sum( A_m[k] * B_m[k] ) / ( ||A_m|| * ||B_m|| + eps )

CosSim = (1/M) * sum(cos_m)    [0..1]
```

Где:
- `||A_m|| = sqrt( sum( A_m[k]^2 ) )` — евклидова норма

---

## 8. STOI (Short-Time Objective Intelligibility Index)

**Функция:** `compute_stoi(reference, test, sr_ref, sr_test, n_fft=512, hop=256, n_bands=15)`
**Единица:** 0..1 | **Направление:** выше — лучше

```
1. Ресемплирование test к sr_ref (линейная интерполяция)

2. Октавные полосы (1/3 октавы):
   f_c[b] = 150 * 2^(b/3),  b = 0, 1, ..., n_bands-1

3. Для каждого фрейма m и полосы b:
   mask = (freq_bins >= f_c - bw/2) AND (freq_bins <= f_c + bw/2)
   env_ref[m,b] = sqrt( mean( |FFT_ref[m, mask]|^2 ) )
   env_tst[m,b] = sqrt( mean( |FFT_tst[m, mask]|^2 ) )

4. Нормировка:
   env_ref = env_ref / (max(env_ref) + eps)
   env_tst = env_tst / (max(env_tst) + eps)

5. Корреляция Пирсона для каждой полосы b:
   r_c = sum( (env_ref - mean(env_ref)) * (env_tst - mean(env_tst)) ) /
         ( sqrt( sum((env_ref - mean(env_ref))^2) * sum((env_tst - mean(env_tst))^2) ) + eps )

6. STOI = ( mean(r_c) + 1 ) / 2    [преобразование [-1,1] -> [0,1]]
```

## 9. PESQ (Perceptual Evaluation of Speech Quality) — приближённый

**Функция:** `compute_pesq_approx(reference, test, sr_ref, sr_test)`
**Единица:** -0.5..4.5 | **Направление:** выше — лучше

```
1. Ресемплирование обоих сигналов к 16 кГц

2. A-взвешивание (упрощённое):
   W(f) = (12194^2 * f^4) / ( (f^2 + 20.6^2) * sqrt( (f^2 + 107.7^2)(f^2 + 737.9^2) ) * (f^2 + 12194^2) )
   W(f) = W(f) / max(W(f))

3. Для каждого фрейма m:
   diff[k] = | log10( (|X_m[k]| + eps) / (|Y_m[k]| + eps) + eps ) |
   D_m = mean( (diff * W)^2 )

4. D = sqrt( mean(D_m) )

5. PESQ = 4.5 - 3.0 * D    [ограничение: [-0.5, 4.5]]
```

---

## 10. MOS (Mean Opinion Score) на основе PESQ

**Функция:** `compute_pesq_mos(reference, test, sr_ref, sr_test)`
**Единица:** 1..5 | **Направление:** выше — лучше

```
MOS = 1 + 3.87 / ( 1 + exp( -1.3669 * (PESQ - 0.7197) ) )    [ограничение: [1.0, 5.0]]
```

---

## 11. Spectral Flatness (Wiener Entropy)

**Функция:** `compute_spectral_flatness(signal, sr, n_fft=2048, hop=512)`
**Единица:** 0..1 | **Направление:** выше — шумоподобнее

```
Для каждого фрейма m:
S[k] = |rFFT( frame_m * w )| + eps

GM = exp( mean( log(S[k]) ) )         (геометрическое среднее)
AM = mean( S[k] )                      (арифметическое среднее)

SF_m = GM / (AM + eps)

SF = (1/M) * sum(SF_m)
```

---

## 12. Dynamic Range

**Функция:** `compute_dynamic_range(signal, percentile_low=5.0, percentile_high=95.0)`
**Единица:** дБ

```
RMS по окнам размером W (шаг W/2):
rms[m] = sqrt( mean( signal[m*W/2 : m*W/2+W]^2 ) )

low = percentile(rms, 5) + eps
high = percentile(rms, 95) + eps

DR = 20 * log10( high / low )    [дБ]
```

---

## 13. Crest Factor

**Функция:** `compute_crest_factor(signal)`
**Единица:** безразмерная (>= 1.0)

```
peak = max( |x[n]| )
rms = sqrt( mean( x[n]^2 ) ) + eps

CF = peak / rms
```

---

## 14. Агрегированный Score

**Функция:** `compute_metrics_batch(...)` — итоговый балл

Min-max нормировка всех метрик к шкале [0, 1] («выше — лучше»):

```
Для метрик «ниже лучше» (LSD, SC, RMSE, SC_diff, Time):
  norm = (max - value) / (max - min + eps)

Для метрик «выше лучше» (SNR, SI-SDR, CosSim):
  norm = (value - min) / (max - min + eps)

Score = 0.20 * LSD_norm +
        0.15 * SC_norm +
        0.20 * SNR_norm +
        0.15 * RMSE_norm +
        0.15 * SI-SDR_norm +
        0.05 * SC_diff_norm +
        0.05 * CosSim_norm +
        0.05 * Time_norm
```
