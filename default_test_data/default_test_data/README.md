# default_test_data

Тестовые WAV-файлы для демонстрации и проверки качества.

Структура:
- Подпапки — условные жанры/категории. Внутри — WAV-треки для пакетной обработки.

Использование:
- В UI кнопка «Запустить набор» рекурсивно собирает `*.wav` из этой папки.
- Результаты сохраняются в `output/`.
